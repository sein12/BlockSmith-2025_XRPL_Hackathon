## 3️⃣ XRPL 활용 설명

이 프로젝트가 **XRP Ledger(XRPL)**과 상호작용하는 방법:

- **사용 기술**

  - `xrpl` TypeScript/Python 라이브러리 사용
  - 구현 기능: Token Escrow, Credential

- **구현 내용**

  1. XRPL의 Credential을 통한 사용자 인증
  2. 조건 충족 시 자동 결제 트랜잭션 생성
  3. XRPL의 Token Escrow 기능으로 자금 예약/송금/취소 처리
